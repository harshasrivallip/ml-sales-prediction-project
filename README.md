# Sales Prediction using Regression

## Project Overview
This project predicts future sales using historical sales data. A machine learning regression model is created to analyze past sales patterns and forecast future sales values. The system helps in understanding sales trends and supports better business decision-making.

## Problem Statement
Predict future sales based on previous sales data using a regression model.

## Workflow
Data Collection → Data Cleaning → Data Analysis → Model Training → Sales Prediction

## Technologies Used
Python, Google Colab, Pandas, NumPy, Matplotlib, Scikit-learn

## Model
Linear Regression is used to learn patterns from past sales data and predict future sales.

## Results
The trained model predicts future sales values and compares them with actual sales using graphs.
